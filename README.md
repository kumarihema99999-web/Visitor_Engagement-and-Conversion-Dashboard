#Visitor Engagement and Conversion Dashboard

 📊 Project Overview
  This Power BI dashboard analyzes website visitor behavior and conversion performance.  
  It combines SQL, Python, and advanced DAX to deliver actionable business insights for digital strategy optimization.

 🔑 Key Features
- KPIs Summary: Bounce Rate, Conversion Rate, Session Duration, Total Page Views  
- Traffic Sources Analysis: Social Media, Direct, Organic, Referral  
- City-Level Segmentation: Top 5 vs. Bottom 5 cities by page views  
- Exit Page Insights: Conversion counts and average session duration by exit page  
- New vs. Returning Visitors: Conversion breakdown for user types  

💡 Business Insights
- Social media drives the highest conversions.  
- Checkout exits show longer session durations, pointing to optimization opportunities.  
- Strong engagement observed from San Antonio, Charlotte, and Austin.  

🛠️ Tech Stack
- Power BI: Dashboard design, advanced DAX measures  
- SQL: Data extraction and calculated tables  
- Python (pandas, NumPy, matplotlib, seaborn): Data preprocessing and visualization
